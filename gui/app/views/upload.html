A properly formatted file has a an extension .csv and has two columns. The first is the timestamp in a MM/DD/YYYY HH:MM, and the second is the wind speed in meters per second. If the file has two wind columns, as in the example image, the second wind column will be ignored. Additionally, the first row is taken as the column headers. The image below serves as an example.
